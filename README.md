# Cartão de Visita Pessoal - IbTech

Projeto 01 da Trilha Frontend: uma página pessoal feita com HTML, CSS e JavaScript puros.

## Aluno

- Bernardo Alvim
- Turma: IbTech Frontend 2026.1

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis, Grid, Flexbox e responsividade
- JavaScript puro com DOM, `localStorage`, Clipboard API e `IntersectionObserver`

## Requisitos atendidos

- HTML semântico com `header`, `nav`, `main`, `section`, `article`, `figure`, `address` e `footer`.
- CSS separado com variáveis, sistema tipográfico, responsividade e estados de interação.
- JavaScript separado com interações:
  - menu responsivo;
  - alternância de tema claro/escuro com persistência;
  - botão para copiar email com feedback temporário;
  - animação de entrada com `IntersectionObserver`;
  - seletor criativo de foco do semestre.
- Conteúdo pessoal com nome, sobre, fotos, vídeo, projetos, contato e elemento criativo.

## Como visualizar

Abra o arquivo `index.html` no navegador.
